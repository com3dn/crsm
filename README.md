# 🛰️ CRSM – Centro Regional de Segurança Marítima

Bem-vindo ao repositório das **Ferramentas CRSM**, um conjunto de utilitários voltados para segurança e operações marítimas.  
Você pode acessar todas as ferramentas diretamente pelo site:

🔗 **[Acesse a Central CRSM](https://com3dn.github.io/crsm/)**

---

## ⚓ Funcionalidades Disponíveis

### 🗺️ Mapa Interativo

🔍 **[Mapa de Casos](https://com3dn.github.io/crsm/mapa_casos.html)**  
Visualize situações diretamente no mapa com interação geográfica. *(Em ajustes)*

---

### ⚙️ Calculadoras

📏 **[Velocidade Média](https://com3dn.github.io/crsm/velocidade.html)**  
Calcule a velocidade de uma embarcação com base em dois horários e distância.

📆 **[Calculadora de ETA](https://com3dn.github.io/crsm/calculadora.html)**  
Estime a data/hora de chegada após uma distância percorrida.

---

### 🌐 Conversores de Coordenadas

🛰️ **[Conversor GMS → Decimal](https://com3dn.github.io/crsm/conversor_gms.html)**  
Converta coordenadas geográficas do formato graus-minutos-segundos para decimal.

📌 **[Conversor Decimal → GMS](https://com3dn.github.io/crsm/conversor_decimal.html)**  
Converta coordenadas decimais para o formato graus-minutos-segundos.

---

### 🚢 Identificação Marítima

🚩 **[Consulta de Bandeira via MMSI](https://com3dn.github.io/crsm/consulta_mmsi.html)**  
Descubra o país de origem de um contato marítimo com base no código MMSI.

---

### ⚓ Portos e Navegação

🧭 **[Distância entre Portos](https://com3dn.github.io/crsm/calculadora_portos.html)**  
Calcule a distância e o tempo estimado entre dois portos.

📡 **[Interceptação de Contato](https://com3dn.github.io/crsm/interceptacao.html)**  
Simule o ponto de interceptação entre duas embarcações em movimento.

📂 **[Cenários de Rotas](https://com3dn.github.io/crsm/cenarios.html)**  
Carregue planilhas com múltiplas rotas para análise e visualização simultânea.

---

### 📝 Comunicação Operacional

📤 **[Formatador de Mensagem Tática](https://com3dn.github.io/crsm/formatador.html)**  
Preencha e exporte mensagens operacionais padrão em `.txt`.

---

## 🎯 Objetivo

Oferecer uma **central acessível e gratuita** com ferramentas de apoio para:

- Monitoramento e previsão de tráfego marítimo
- Coordenação de interceptações e escoltas
- Cálculo de rotas e desempenho naval
- Criação de mensagens padronizadas táticas

---

## 🧰 Tecnologias Utilizadas

- HTML + CSS + JavaScript
- Leaflet.js (mapas)
- XLSX.js (planilhas)
- Estrutura modular com arquivos `.json` e formulários dinâmicos
